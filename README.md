# Сайт прогнозов на матчи ПО КХЛ

Мини-приложение на Flask для компании друзей:
- личный кабинет пользователя (профиль, статистика, место в таблице);
- отдельная страница текущих прогнозов;
- отдельная страница результатов и лидерборда;
- админ-функции для управления парами ПО и внесения итогов.

## Роли
- **Обычный пользователь**: профиль + прогнозы + просмотр результатов.
- **Админ**: те же возможности, что у обычного пользователя, плюс:
  - `/admin/matches` — задавать пары (конференция Восток/Запад, стадия, дата);
  - `/admin/results` — проставлять финальные счета.

Админ создается при регистрации, если ввести корректный `admin_key`.
По умолчанию ключ: `OMSK` (можно поменять через переменную окружения `ADMIN_KEY`).

## Логика начисления очков
Очки считаются по сложной схеме + вес стадии плей-офф.
На странице результатов правила скрыты под спойлером.

## Быстрый запуск (локально)
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Быстрый деплой на VPS через Docker
```bash
git clone https://github.com/R0uxgar0ux/PO_KHL.git
cd PO_KHL
docker compose up -d --build
```

Сайт будет доступен по адресу `http://<VPS_IP>:5000`.

Полезные команды:
```bash
docker compose logs -f
docker compose down
docker compose up -d --build
```
