# Сайт прогнозов на матчи ПО КХЛ

Мини-приложение на Flask, где друзья могут:
- регистрироваться и входить в личный кабинет;
- оставлять прогнозы на матчи плей-офф КХЛ;
- смотреть текущие результаты, очки и общий рейтинг участников;
- фиксировать реальные итоги матчей через страницу `/admin/results` (для пользователя `admin`).

## Логика начисления очков (продвинутая)
Для каждого прогноза считается база, потом умножается на вес стадии плей-офф.

### Базовые очки
- **+2** — угадан исход (победа/поражение/ничья).
- **+2** — угадана разница шайб.
- **+1** — угадана суммарная результативность (тотал).
- **+4** — точный счет.
- **+1** — риск-бонус за высокий тотал (если прогноз и факт 7+ шайб и почти совпали по тоталу).

### Вес стадии ПО
- `R1` (1/8 финала) — **x1.00**
- `QF` (1/4 финала) — **x1.25**
- `SF` (1/2 финала) — **x1.60**
- `F` (финал) — **x2.20**

Итог матча: `round(base_points * stage_weight)`.

## Быстрый старт (локально)
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После запуска откройте: `http://127.0.0.1:5000`.

## Быстрый деплой на VPS через Docker
На сервере (с установленными Docker и Docker Compose):

```bash
git clone <your_repo_url>
cd PO_KHL
docker compose up -d --build
```

После запуска сайт будет доступен на `http://<VPS_IP>:5000`.

Полезные команды:
```bash
docker compose logs -f
docker compose down
docker compose pull && docker compose up -d --build
```

База хранится в папке `./data` на хосте (смонтирована в контейнер).

## Админ для внесения результатов
1. Зарегистрируйте пользователя с логином `admin`.
2. Войдите под ним.
3. Перейдите на `/admin/results` и введите счет завершенных матчей.
