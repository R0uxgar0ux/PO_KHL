# Сайт прогнозов на матчи ПО КХЛ

Мини-приложение на Flask для компании друзей:
- личный кабинет пользователя (профиль, статистика, место в таблице);
- отдельная страница текущих прогнозов;
- отдельная страница результатов и лидерборда;
- отдельная страница дерева ПО с сериями и счетами;
- админ-функции для управления парами, результатами и пользователями.

## Роли
- **Обычный пользователь**: профиль + прогнозы + просмотр результатов/дерева.
- **Админ**: те же возможности, плюс:
  - `/admin/matches` — добавлять и удалять пары (конференция Восток/Запад, стадия, дата);
  - `/admin/results` — проставлять финальные счета;
  - `/admin/users` — блокировать, разблокировать, удалять простых пользователей.

Админ создается при регистрации, если ввести корректный `admin_key`.
По умолчанию ключ: `OMSK` (можно поменять через переменную окружения `ADMIN_KEY`).

## Валидация имен
- Логин и отображаемое имя: только латиница/цифры/`_`, длина 3–24.
- Отображаемое имя уникально и используется в общей таблице результатов.

## Быстрый запуск (локально)
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Быстрый деплой на VPS через Docker
```bash
git clone https://github.com/R0uxgar0ux/PO_KHL.git
cd PO_KHL
docker compose up -d --build
```

Сайт будет доступен по адресу `http://<VPS_IP>:5000`.

Полезные команды:
```bash
docker compose logs -f
docker compose down
docker compose up -d --build
```
